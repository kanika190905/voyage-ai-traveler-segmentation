# Voyage AI - Traveler Segmentation

This project uses machine learning to analyze airline customer behavior and segment travelers into meaningful groups.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- K-Means Clustering
- PCA
- Matplotlib

## Project Workflow
1. Data preprocessing and cleaning
2. Feature scaling using StandardScaler
3. Applying K-Means clustering
4. Finding optimal clusters using the Elbow Method
5. Dimensionality reduction using PCA
6. Visualizing traveler segments

## Goal
To identify patterns in traveler behavior so airlines can design better marketing and customer engagement strategies.
